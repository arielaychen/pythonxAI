# ⚖️ 1. 比較運算子（比大小）

比較運算子可以比較兩個東西，結果只有兩種：

- **True（對、成立）**
- **False（錯、不成立）**

| 符號 | 意思     | 範例   | 結果 |
| ---- | -------- | ------ | ---- |
| ==   | 等於     | 5 == 5 | True |
| !=   | 不等於   | 5 != 3 | True |
| >    | 大於     | 8 > 5  | True |
| <    | 小於     | 3 < 8  | True |
| >=   | 大於等於 | 5 >= 5 | True |
| <=   | 小於等於 | 4 <= 6 | True |

### 範例

```python
print(5 == 5)
print(5 > 8)
print(3 != 3)
```

畫面會顯示：

```
True
False
False
```

💡 **小提醒：** 比較時，最好是相同種類的資料互相比較，例如數字跟數字、文字跟文字。

# 🧠 2. 邏輯運算子（一起判斷）

有時候需要一次判斷很多條件，就可以使用邏輯運算子。

## （1）and（而且）

意思是：

**全部都要是真的，結果才會是 True。**

就像：

👉 今天要出去玩，**要天氣好，而且作業寫完。**

兩個都做到才能出去玩。

### 範例

```python
print(True and True)
print(True and False)
```

結果：

```
True
False
```

---

## （2）or（或者）

意思是：

**只要有一個是真的，就算成功。**

例如：

👉 放假可以去公園 **或** 去圖書館。

只要其中一個可以去，就可以。

```python
print(True or False)
print(False or False)
```

結果：

```
True
False
```

---

## （3）not（相反）

not 會把結果反過來。

```python
print(not True)
print(not False)
```

結果：

```
False
True
```

# 🚦 4. if 判斷式（讓程式做選擇）

程式可以根據不同情況做不同事情。

例如：

```python
score = 90

if score >= 60:
    print("及格")
else:
    print("不及格")
```

如果成績大於等於 60，就會顯示：

```
及格
```

---

## if、elif、else

可以判斷很多種情況。

```python
if 分數 >= 90:
    A
elif 分數 >= 80:
    B
elif 分數 >= 70:
    C
else:
    不及格
```

程式會：

1. 先看第一個條件。
2. 如果不是，再看第二個。
3. 找到符合的就停止。
4. 都不符合才會執行 `else`。

# 🔁 11. for 迴圈（重複做事情）

有時候同一件事情要做很多次，就可以使用 **for 迴圈**。

例如：

```python
for i in range(5):
    print(i)
```

結果：

```
0
1
2
3
4
```

因為：

```
range(5)
```

代表：

```
0、1、2、3、4
```

**不包含 5。**

# ✏️ 13. 迴圈中的變數

```python
for i in range(5):
    a = i * 2
```

每一次：

| i   | a   |
| --- | --- |
| 0   | 0   |
| 1   | 2   |
| 2   | 4   |
| 3   | 6   |
| 4   | 8   |

最後：

```python
print(a)
```

只會印出：

```
8
```

因為 `a` 每次都被新的數值蓋掉，最後留下的是最後一次迴圈的結果。
