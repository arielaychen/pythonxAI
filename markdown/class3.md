# Python 上課筆記（國小版）

## 1. 什麼是 Streamlit？

Streamlit 是一個可以把 Python 程式變成網頁的小工具。

只要寫一些程式，就可以做出：

- 按鈕
- 輸入框
- 數字選擇
- 漂亮的畫面

---

# 2. 輸入數字：number_input

```python
number = st.number_input("請輸入一個數字", step=1, min_value=1, max_value=9)
```

### 功能

讓使用者輸入一個數字。

### 常用設定

- `step=1`：一次增加或減少 1。
- `min_value=1`：最小只能輸入 1。
- `max_value=9`：最大只能輸入 9。

例如：

如果輸入 **5**

程式就會知道：

```
number = 5
```

---

# 3. 顯示文字

```python
st.markdown("文字")
st.write("文字")
```

可以把文字顯示在畫面上。

例如：

```python
st.markdown(f"你輸入的是 {number}")
```

畫面會看到：

```
你輸入的是 5
```

---

# 4. for 迴圈（重複做事情）

```python
for i in range(1, number+1):
```

意思就是：

「從 1 開始，一直做到 number。」

例如：

```
number = 5
```

就會變成

```
1
2
3
4
5
```

---

## 範例

```python
st.markdown(str(i)*i)
```

如果 i 是：

```
1
11
222
3333
44444
```

因為

```
str(i)
```

會把數字變成文字。

# 5. List（串列）

List 就像一個盒子，可以放很多東西。

例如：

```python
L=[1,2,3]
```

盒子裡有：

```
1
2
3
```

也可以放不同種類：

```python
L=[1,"a",True,3.14]
```

可以放：

- 數字
- 文字
- True/False
- 小數

甚至還可以放另一個 List。

---

# 6. List 的位置（Index）

每個東西都有自己的位置。

注意：

**Python 從 0 開始算！**

例如：

```python
L=["蘋果","香蕉","葡萄"]
```

位置如下：

| 位置(Index) | 內容 |
| ----------- | ---- |
| 0           | 蘋果 |
| 1           | 香蕉 |
| 2           | 葡萄 |

所以：

```python
L[0]
```

得到

```
蘋果
```

---

# 7. List 切片（Slice）

可以一次拿很多資料。

例如：

```python
L=[1,2,3,4,5]
```

### 全部

```python
L[:]
```

得到

```
[1,2,3,4,5]
```

---

### 每隔一個拿一次

```python
L[::2]
```

得到

```
[1,3,5]
```

---

### 拿第1到第3個位置

```python
L[1:4]
```

得到

```
[2,3,4]
```

注意：

最後一個位置 **不包含**。

---

# 8. append（加入資料）

```python
L.append(4)
```

原本

```
[1,2,3]
```

變成

```
[1,2,3,4]
```

就是把新東西放到最後。

---

# 9. remove（刪除指定內容）

```python
L.remove("a")
```

如果

```
["a","b","a"]
```

只會刪掉第一個

```
["b","a"]
```

---

# 10. pop（刪除指定位置）

```python
L.pop(0)
```

就是刪掉第 0 個。

如果

```
["a","b","c"]
```

會變成

```
["b","c"]
```

如果沒有寫位置：

```python
L.pop()
```

就是刪掉最後一個。

---

# 11. sort（排序）

```python
L.sort()
```

例如

```
[3,1,5,2]
```

變成

```
[1,2,3,5]
```

會直接改變原本的 List。

---

# 12. 走訪 List（把每個東西都看一次）

方法一：

```python
for i in L:
```

一次拿一個資料。

例如

```
蘋果
香蕉
葡萄
```

都會被印出來。

---

方法二：

```python
for i in range(len(L)):
```

可以知道位置（Index）。

例如：

```
第0個
第1個
第2個
```

# 13. Columns（分欄）

可以把畫面切成很多格。

例如：

```python
col1,col2=st.columns(2)
```

畫面變成：

```
□    □
```

左右兩欄。

也可以：

```python
st.columns(4)
```

變成四欄。

---

## 比例分欄

```python
st.columns([1,2])
```

表示：

```
□   □□□□□
```

右邊比左邊大。

---

# 14. with

```python
with col1:
```

表示：

「接下來放的東西都放進 col1。」

例如：

```python
with col1:
    st.button("按鈕")
```

按鈕就會出現在第一欄。

---

# 15. Button（按鈕）

```python
st.button("按鈕")
```

畫面會出現按鈕。

按下去後，可以做事情。

例如：

```python
if st.button("按我"):
    st.balloons()
```

按下去會放氣球🎈。

# 16. Text Input（文字輸入）

```python
text=st.text_input("請輸入文字")
```

可以輸入文字。

例如：

```
你好
```

程式就會知道：

```

```

---

# 17. session_state（記住資料）

一般變數：

```python
ans=1
```

每次重新整理都會變回

```
1
```

所以會忘記之前的數字。

---

使用：

```python
st.session_state
```

就可以把資料記住。

例如：

```
1
↓

2

↓

3

↓

4
```

按按鈕後，數字會一直增加，不會重新變回 1。

---

# 18. st.rerun()

```python
st.rerun()
```

意思是：

**重新執行整個程式。**

有時候畫面沒有更新，就可以使用它。
